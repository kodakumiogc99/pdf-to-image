# PDF to Image Converter

- 給定 pdf 檔案，可以轉成圖片
- 輸出結果預設 jpg，可支援多種輸出格式
- 可支援 batch 轉換

## 專案架構

- 用 `uv` 管理專案並用 hatchling 建置環境
- 程式碼請放 `src/pdf_to_image` 資料夾下
- `pdf_to_image` 底下有 `core`, `qt`, `streamlit`, `apis` 資料夾
- 所有算法、功能都放 `core` 下
- `qt` 放跟 `pyqt` 有關的單機介面
- `streamlit` 和 `apis` 先不要實作

## 需求

- 將pdf轉成圖片
- 可以支援多種格式的輸出，預設 jpg，使用者可透過下拉選單選擇輸出
- 可支援 batch 轉換
- 使用者透過拖拉或點擊上傳，可以一次選取多個檔案或資料夾，如果是資料夾要recursively的拜訪該資料夾內的所有pdf檔案
- 上傳的檔案只能是 pdf 或資料夾
- 可設定輸出路徑，預設本專案下的 `outputs` 資料夾
- 所有使用者的設定都要被記錄下來，包含上次上傳的路徑
- **背景執行緒 (QThread)**：轉換過程不造成介面凍結，並支援多頁面進度計算。
- **進度條顯示**：介面即時顯示批次轉換的整體進度。
- **日誌系統 (Logging)**：過程及錯誤詳細記錄於根目錄下的 `app.log` 中。
